# Teal Generator

Генератор цветовых токенов для платформы Teal HR.  
Позволяет создать полную систему цветов на основе брендового цвета и сразу увидеть результат.

## Возможности
- 🎨 Генерация 30+ цветовых токенов из одного primary-цвета
- 🔄 Три алгоритма: Default, Monochromatic, Saturation
- 🖼️ Загрузка атрибутов бренда (логотип, аватар, иконки, баннер)
- 👀 Превью платформы в реальном времени
- 📦 Экспорт проекта в ZIP с токенами и всеми атрибутами

## Запуск проекта

```bash
npm install     # установка зависимостей
npm run dev     # запуск в режиме разработки
npm run build   # сборка для продакшена
