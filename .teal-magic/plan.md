
## Генератор цветовых токенов для кастомизации платформы

### Что делает инструмент
Вы вводите один основной (brand) цвет → инструмент автоматически генерирует полную палитру из ~30+ токенов → показывает превью UI-элементов → скачивает готовый JSON-файл для разработки.

---

### Экран 1: Ввод цвета и генерация

**Панель ввода (левая часть):**
- Поле выбора основного цвета (color picker + HEX-ввод)
- Опциональные поля для статусных цветов (warning, danger, success) с дефолтными значениями, которые можно переопределить
- Кнопка «Сгенерировать палитру»

**Правила генерации оттенков из одного цвета:**
- **Background accent** — основной цвет
- **Background island-inner** — очень светлый оттенок основного цвета (5-10% насыщенности)
- **Background globe/island** — почти белый/белый
- **Background border** — нейтральный серый, слегка окрашенный в тон основного
- **Background modal** — чёрный с прозрачностью 60%
- **Text primary** — тёмный нейтральный цвет на основе оттенка
- **Text secondary/tertiary** — серые с градацией
- **Text link** — основной цвет
- **Text inverse** — белый и светлые оттенки
- **Icons primary** — основной цвет
- **Buttons primary** — основной цвет + hover (светлее на 10-15%) + active (темнее на 10-15%) + disabled (40% прозрачности)
- **Buttons outline** — белый фон, основной цвет для текста/бордера, hover/active/disabled варианты
- **Buttons inverse** — белый фон, основной цвет для текста
- **Shadow** — стандартная тень

### Экран 2: Превью и результат

**Панель превью (правая часть):**
- Визуальное превью сгенерированных UI-элементов: кнопки (primary, outline, inverse) в состояниях default/hover/active/disabled, текстовые блоки (primary, secondary, tertiary), ссылки, иконки, фоновые блоки (globe, island, island-inner, accent), статусные бейджи (warning, danger, success)
- Цветовые плашки с RGB-значениями для каждого токена

**Действия:**
- Возможность подправить любой отдельный цвет вручную после генерации
- Кнопка «Скачать JSON» — сохраняет файл в точном формате вашего шаблона (с секциями background, text, icons, status, buttons)
- Кнопка «Сбросить» — очистить и начать заново

---

### Технические детали
- Чисто клиентское приложение, без бэкенда
- Генерация оттенков через HSL-манипуляции (lightness, saturation)
- JSON выгружается точно в формате ваших существующих файлов (tokens1-8.json)
